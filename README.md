# JjMonetizacion

## Instalación
1. Requisitos.
- Node.js
- npm
- angular-cli
2. Clonar este repositorio.
3. En la raiz del projecto ejecutar "npm install".

## Modo desarollador
Ejectuar `ng serve` para modo desarollador.
Direccón por defecto "http://localhost:4200/"

## Modo produccion
Ejectuar `ng build --prod` para modo desarollador.
